🎵 CMFI Choir Website

Welcome to the Christian Missionary Fellowship International (CMFI) Choir Website.

This website was created to showcase the ministry of the CMFI Choir, promote worship, share upcoming events, and provide information about the choir's mission and activities.

- Home page with welcome section
